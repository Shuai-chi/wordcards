# WordForge - 高品質英語單字學習系統 🚀

一個專為進階英語學習者設計的極簡單字卡系統。支援 PWA、自定義 CSV 匯入，並具備精確的 SRS (間隔重複) 複習機制。

## ✨ 特色功能
*   **高品質詞庫**：支援自定義 CSV 匯入，並具備精確的 SRS 複習機制。
*   **智慧清洗**：自動過濾 AI 冗餘語句與 fillers。
*   **多語境支援**：完美適配 Academic, Campus 與 Daily 不同場景。

## 📲 安裝與存取方式

### 1. 網頁版直接使用 (最快)
本專案為純前端應用，您可以直接開啟部署好的網頁 URL。
*   **離線支援**：只要開啟過一次，後續即使無網路也能使用。
*   **資料儲存**：所有單字卡與學習紀錄皆儲存在您瀏覽器的本地資料庫 (IndexedDB) 中，不會上傳到任何伺服器。

### 2. 手機端安裝 (PWA 推薦)
為了獲得最佳體驗（全螢幕、更順暢的語音播放），強烈建議安裝為 **PWA 應用程式**：
*   **iOS (Safari)**：點擊底部的「分享」按鈕 -> 選擇 **「加入主畫面」**。
*   **Android (Chrome)**：點擊右上角選單 -> 選擇 **「安裝應用程式」**。

### 3. 本地開發安裝
1. `git clone https://github.com/your-repo/wordforge.git`
2. `npm install`
3. `npm run dev`

## 🛠️ 使用教學

### 第一步：準備與匯入單字卡
1.  **產出單字卡**：參考 [單字卡規範與 AI 提示詞](./prompts/SPEC.md) 來產出高品質 CSV。
2.  **匯入**：在首頁點擊右上角的 **「上傳」** 按鈕，選擇您的 CSV 檔案。

### 第二步：學習設定
1.  **全域每日上限**：點擊右上角「設定」，設定全域每日新卡片上限。
2.  **單一分組上限**：每個匯入的套牌可以單獨設定每日新卡片數量（例如：科技單字 20 張、日常單字 10 張）。

### 第三步：開始練習 (SRS 機制)
系統會根據您的反饋決定卡片下次出現的時間：
*   **Again (1)**：完全不記得。卡片會進入重學流程，並在短時間內再次出現。
*   **Hard (2)**：認識單字但反應較慢。縮短下次複習的間隔。
*   **Good (3)**：能順利辨識。使用標準的間隔增長。
*   **Easy (4)**：毫不費力。大幅拉長下次複習的間隔。

### 第四步：重複練習
若當日任務已完成，但仍想複習：
*   首頁按鈕會變更為 **「重複練習今日卡片」**，讓您能夠針對當天練習過的內容進行鞏固，此操作不會影響 SRS 的長期排程。

## 📂 專案結構
*   `src/components/`：UI 組件（包含 Dashboard 與 LearningView）。
*   `src/lib/`：核心邏輯（SRS 演算法、資料庫操作）。
*   `prompts/`：包含 AI 提示詞範例與單字卡 CSV 格式規範。

## 🛡️ 隱私聲明
WordForge 是一個 **「私有化」** 工具。您的所有學習數據、單字內容均保存在本地瀏覽器緩存中。這意味著您的隱私受到絕對保護，但也提醒您定期導出或備份重要單字庫。
